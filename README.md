# Mobili-JSON Server

Banco de dados simples para testes com o react.

## Instalação

Clone o repositorio:

```bash
  git clone https://github.com/CO-DE-Mobili/Mobili-JSON_Server.git
  cd Mobili-JSON_Server
```

Instale o pacote:

```
    npm install -g json-server
```

Inicie o servidor:

```
    json-server --watch db.json
```
